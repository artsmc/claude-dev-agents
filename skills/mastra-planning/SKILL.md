# Mastra Planning & Team Orchestration

Strategic planner that analyzes Mastra development tasks, classifies complexity, selects optimal team compositions, and creates structured execution plans with task dependencies. Routes to the right specialized Mastra skills and agents.

## Usage

```bash
/mastra-planning
```

Provides context for:
- Domain detection (agents, workflows, RAG, memory, evals, streaming, deploy, MCP, workspace)
- Complexity classification (Level 1-3)
- Team composition templates (7 pre-built templates)
- Task breakdown with parallel execution groups
- Skill injection into agent prompts
- Decision framework for choosing patterns
- Integration with all 10 Mastra skills
