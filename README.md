# Claude Dev Agents

Custom agents and commands for Claude Code (claude.ai/code).

## Structure

- `agents/` - Custom Claude agents for specialized tasks
- `commands/` - Slash commands for Claude Code
- `CLAUDE.md` - Global instructions for Claude Code

## Agents

### Code Review & Development
- **code-reviewer.md** - Automated code review agent
- **frontend-developer.md** - Frontend application logic and state management
- **ui-developer.md** - Visual implementation and styling
- **nextjs-backend-developer.md** - Next.js backend development
- **nextjs-qa-engineer.md** - Quality assurance and testing for Next.js

## Commands

### Productivity Tools
- **document-hub.md** - Documentation management
- **memory-bank.md** - Knowledge storage and retrieval
- **start-phase.md** - Project phase initialization

## Usage

These agents and commands are automatically available in Claude Code when placed in the `~/.claude/agents/` and `~/.claude/commands/` directories respectively.
